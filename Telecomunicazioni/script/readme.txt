Questo è un piccolo progetyto realizzato due giorni prima dell'esame di Telecomunicazioni per esercitarsi
con i possibili test di Telecomunicazioni.
Per installare tutti i pacchetti necessari (Pillow per la gestione delle immagini) utilizzare: 

$ pip install Pillow      #Mettere il path di requirements.txt se non siete all'interno della sua cartella quando si esegue il comando pip

Se si è con un SO normodotato dovrebbe funzionare facilmente, mentre se siete su Arch Linux come quel deficiente di Spina,
controllare sulla AUR per i download dei pacchetti (dovrebbe essere:
$ pipx install Pillow
,però non lo so per certo)

Le soluzioni dei quiz non sono state verifiate, quindi non prendetele come delle assolute verità.
